# Vercel 部署说明

这是已经转换好的 Vercel 版 Next.js 项目。

## 推荐部署方式

1. 解压 `autaksing-resume-vercel.zip`。
2. 把解压后的整个项目文件夹上传到 GitHub。
3. 在 Vercel 选择 **Add New → Project**，导入该 GitHub 仓库。
4. Framework Preset 保持 **Next.js**，其他设置使用默认值。
5. 点击 **Deploy**。

项目不需要数据库、环境变量或额外密钥。

## 注意

Vercel 控制台通常不支持直接上传 ZIP 作为项目，最稳妥的方式是先解压并上传到 GitHub，再由 Vercel 导入仓库。
